# Opening-Website Specification

## Overview
An animated beach scene personal introduction page featuring sky, sun, clouds, and ocean waves.

## Features
- **Animated Sun**: Pulsing golden sun with glow effect
- **Moving Clouds**: Two clouds drifting across the sky
- **Ocean Waves**: Three layered wave animations with varying opacity
- **Personal Intro**: Name and greeting message centered on screen

## Visual Design
- **Sky**: Light blue gradient (`#87CEEB`)
- **Beach**: Wheat color (`#f5deb3`)
- **Ocean**: Deep blue gradient (`#006994` to `#004e7a`)
- **Typography**: Georgia serif font, white text with shadow

## Animations
| Element | Animation | Duration |
|---------|-----------|----------|
| Sun | Pulse/scale | 3s infinite |
| Cloud 1 | Move left to right | 25s linear |
| Cloud 2 | Move left to right | 30s linear |
| Waves | Horizontal scroll | 8-12s linear |

## Technical Details
- **File**: `index.html`
- **Framework**: Vanilla HTML/CSS
- **Responsive**: Flexbox centered layout, 100vh height
